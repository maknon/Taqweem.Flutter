awqat.cc

- Set Windows title
	- Edit 'windows\runner\main.cpp'
	- Replace the title in 'window.Create(...'
	- Make sure that the file fortmat is UTF-8-BOM
	
- Set MacOS title
	- Edit 'macos\Runner\Configs\AppInfo.xcconfig'
		- Change 'PRODUCT_NAME'
	- Edit 'macos\Runner\Release.entitlements' and add:
		<key>com.apple.security.network.client</key>
		<true/>

- Set ios title
	- Edit 'ios\Runner\Info.plist'
	- Change 'CFBundleName' & 'CFBundleDisplayName'
	- Add:
	    <key>NSMicrophoneUsageDescription</key>
        <string>The app does not use this feature</string>

- Set Linux title
	- Edit 'linux\my_application.cc'
	- Replace the title in 'gtk_header_bar_set_title(...' & 'gtk_window_set_title(...'

- Set Web title
	- Edit 'web\index.html'
	- Change <title>

- Set Android title
	- Edit 'AndroidManifest.xml'
	- Change 'android:label'

- Windows binaries is built using 'build_win.bat', no menu in Android Studio for it
- Linux binaries is built using 'build_linux.sh', no need for Android Studio in linux

- Icons are generated by https://icon.kitchen/, it should be run on Edge as svg is not running in Firefox
	- icon.svg is used as a main src
	- icon_web.svg is another option to be used for all small icons
	- Follow README.txt for web inside the IconKitchen-Output.zip
	- ios has 1024 x 1024 (AppIcon~ios-marketing.png), use it and generate the rest of macos icons from https://www.appicon.co/. Use 512 from android play_store_512.png
	- for windows pick from ios folder 120 x 120 and use gimp to generate ico file
	- Locations:
		- macos\Runner\Assets.xcassets\AppIcon.appiconset
		- web\icons
		- windows\runner\resources
		- android\app\src\main\res
		- ios\Runner\Assets.xcassets\AppIcon.appiconset
